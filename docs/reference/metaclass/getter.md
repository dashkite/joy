# getter

_Function_

Defines a getter on _target_ using the given _key_ (the property name) and _description_. Like most Joy functions, &#x60;getter&#x60; is curryable, so you can define a function that will define a property on any object. Use in combination with [&#x60;mixin&#x60;](#mixin) to create reusable mixins.

<pre><code>getter key, function, target &rarr; object</code></pre>
<br>

| name | type | description |
|------|------|-------------|
|key|any||
|function|any||
|target|any||
|object|any||
