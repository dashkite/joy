# splice

_Function_ &bull; Removes the elements in _array_ from _start_ up to, but not including, _end_, and returns the mutated array.

<pre><code>splice start, end, array &rarr; array</code></pre>
<br>

| name | type | description |
|------|------|-------------|
|start|any||
|end|any||
|array|any||
|array|any||
